# ETL_Pipeline

A Python ETL pipeline project.

## Quick Start

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Add local environment values in `.env` (do not commit secrets).

## Project Setup for GitHub

This repository includes:
- A Python-focused `.gitignore`
- A starter `.env.example`

## Push to GitHub

Run these commands locally after creating an empty GitHub repository:

```bash
git add .
git commit -m "Initial project setup"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```
